# COSC_241_CHECKERS_FINAL

Applying AI algorithms to solve the game of checkers.

MCTS_v_agents.py includes Eric's code for playing against a Custom Agent.

adv_agents.py includes codes for the Custom Agent and the Random Agent.

main.py runs tests on the mcts and mctsClass files.

mcts.py is Bryan's original implementation of MCTS on py-draughts.

mctsClass.py is Andrew's interpolation of Bryan's code into a class structure.

node.py contains the code for the node class used in mcts and mctsClass.

requirements.txt has the required libraries (just pydraughts).
