# COSC_241_CHECKERS_FINAL

Applying AI algorithms to solve the game of checkers.

(Eric) MCTS_v_agents.py includes code for playing against a Custom Agent.  
(Andrew) MCTS_v_random.py runs tests against a Random Agent.  
(Eric) adv_agents.py includes code for the Custom Agent and the Random Agent.  
(Bryan) main.py runs tests on the mcts and mctsClass files.  
(Bryan) mcts.py is Bryan's original implementation of MCTS on py-draughts.  
(Andrew) mctsClass.py is Andrew's interpolation of Bryan's code into a class structure.  
node.py contains the code for the node class used in mcts and mctsClass.  
requirements.txt has the required libraries (just pydraughts).
